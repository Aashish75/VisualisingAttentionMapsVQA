Visualizing Attention Maps as Evidence for Visual Question Answering

Guidelines:

  • We recommend using the default image to begin with (it's small and quick to process).
  • Type some relevant text in the input field.
  • Click on Process Image and wait for the results.
  • If you don't see the Event ID or the output visualization, the backend app may still be initializing. It takes upto 30 seconds for the backend to initialize.
  • This uses a Grad-CAM based approach for visualizing attention in Visual Question Answering.
  • Backend runs on Hugging Face Spaces (gradio app) and frontend is built with HTML, CSS, and JavaScript.
